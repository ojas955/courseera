# Contributing

All contributions, bug reports, bug fixes, documentation improvements, enhancements, and ideas are welcome.

## How to Contribute

- Open an issue to discuss your idea or report a bug.
- Fork the repository and create a feature branch.
- Keep changes focused and include clear commit messages.
- Submit a pull request with a summary of your changes.

## Code Style

- Keep scripts simple and readable.
- Prefer clear variable names and consistent formatting.

## Community

Please follow the Code of Conduct when participating in this project.
